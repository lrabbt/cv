# Curriculum Vitae

Template from [LaTeX Templates](https://www.latextemplates.com/template/developer-cv)

## Building from source

Requires:

* TexLive
* TexLive's LaTeX packages

Cloning:

```sh
git clone git@github.com:lrabbt/cv.git
cd cv
```

Building:

```sh
pdflatex curriculum-vitae
```
